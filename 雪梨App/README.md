# 雪梨App (Xueli App)

这是一个互动式阅读与角色对话平台的项目。

## 项目结构

- **PRD.md**: 产品需求文档，包含详细的功能规划、用户角色、商业模式及UI/UX设计规范。

## 简介

雪梨App旨在连接内容创作者与读者。读者不仅可以阅读优质故事，还能与故事中的角色进行实时AI对话，获得沉浸式的陪伴体验。

## 核心功能

- **作者端**: 作品发布、角色人格设定 (Persona Setting)。
- **读者端**: 沉浸式阅读、AI角色伴侣、亲密度养成。
- **商业化**: 章节付费、对话付费、虚拟礼物。

详细规划请参阅 [PRD.md](./PRD.md)。
